# Computer Networks Lab Tasks
### Overview

This repository contains a series of lab tasks demonstrating various aspects of computer networking using Cisco Packet Tracer. Each task involves configuring network components like PCs, switches, routers, and DHCP servers. The labs cover different scenarios such as direct PC-to-PC connections, switch-router setups, DHCP configurations, wireless networks, and more.

For detailed steps, images, and configurations, refer to the [full report](./Report.pdf) included in this repository.

### Requirements

To run these labs, you will need:

Cisco Packet Tracer (version 8.2.2.0400)

Basic knowledge of networking concepts (IP addresses, subnetting, routing)

### How to Use

Clone this repository:

git clone https://github.com/yourusername/network-labs.git


Open the .pkt files in Cisco Packet Tracer.

Follow the lab steps to configure each network.

Verify connectivity by sending packets or pinging devices.
